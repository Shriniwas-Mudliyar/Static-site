Welcome to My AWS Portfolio

This website is a personal project I built to practice and showcase my cloud and web development skills. It is hosted on AWS S3 as a static website and distributed globally with AWS CloudFront for performance and security.

Key highlights of this project:

  • 🚀 Fully deployed with AWS S3 + CloudFront
  • 🔐 Secured using Origin Access Control (OAC)
  • 📑 Includes dedicated pages: Home, About, Projects, Quotes, Contact
  • 🎨 Clean, responsive design with a blue & white theme
  • 🌍 Built to demonstrate my AWS Cloud Practitioner skills

This portfolio will continue to grow as I add more cloud projects and experiment with new AWS services.
